abstract: 'A key problem confronting theories of false memory is that false-memory
  phenomena are so diverse: Some are characteristic of controlled laboratory tasks,
  others of everyday life; some occur for traumatic events with legal consequences,
  others for innocuous events; some are characteristic of one developmental level,
  others of another developmental level. Fuzzy-trace theory explains false memories
  via a small set of principles that implement a single representational distinction.
  Those principles generate new predictions, some of which are counterintuitive.'
